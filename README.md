# Bloček
Je taková malá aplikace. Jedná se jen o takový hrubý základ pro osobní kalendář. Aplikace zobrazuje měsíční kalendář, do kterého je možné ukládat různé poznámky.

Java SE 10.0.1

## IDE

IntelliJ IDEA

## Knihovny
